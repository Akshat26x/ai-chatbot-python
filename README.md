# AI Chatbot (Python)

## Overview
This is a simple chatbot built using Python that interacts with users and responds to basic queries.

## Features
- Interactive command-line chatbot
- Handles multiple user queries
- Simple rule-based responses

## Tech Stack
- Python

## How to Run
1. Open terminal
2. Run:
   python app.py
